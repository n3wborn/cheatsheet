### Select content of an element

```bash
CTRL+SHIFT+A
```


### Wrap selection in HTML tag

```bash
ALT+SHIFT+W
```


### Toggle clipboard

```bash
CTRL+K+V
```


### Toggle sidebar

```bash
CTRL+K+B
```


### Select line

```bash
CTRL-L
```


### Select duplicates

```bash
CTRL+D
```
